# Hm... Die Auswahl einer Programmiersprache?

## Bevor wir das tun

Du solltest zumindest eine vage Idee haben was du mit Programmieren als Fähigkeit erreichen oder ausprobieren willst (z.B. eine spezifische Programmiersprache, Technologie oder ein bestimmter Bereich).

## Verschiedene Sprachen, Verschiedene Anwendungen

Jede Programmiersprache kann mehr oder weniger geeignet für ein bestimmtes Einsatzgebiet sein, je nach dem was sie dir ermöglicht oder was für ein Ökosystem du rund um die Sprache erwarten kannst.

> Merke:
>
> Neuere Sprachen mögen mit Vorteilen wie erhöhten Geschwindigkeiten oder einer verinfachten Nutzung einhergehen, jedoch fehlt ihnen oft das etablierte Ökosystem, das ältere Sprachen bieten.
>
> Du musst die Vor- und Nachteile abwägen, wenn du eine Programmiersprache auswählst, die du lernen bzw. in der du ein Programm schreiben willst.

## Also welche Sprachen werden häufig gewählt?

Obwohl die Auswahl deiner Programmiersprache oft mit deinen Zielen in Bezug auf Programmierung zusammenhängt, gibt es ein paar offensichtliche Kandidaten:

<details>
  <summary>
    Anfangssprachen, die man kennen sollte:
  </summary>
  <details>
    <summary>
    1. Python
    </summary>
      Der Schatz der Anfängersprachen. Flexibel, einfach zu lernen und fantastisch zum Experimentieren.<br />
      Dies ist eine offensichtliche Wahl für die Unentschlossenen unter euch, zusammen mit den Leuten, die Python für ein Projekt nutzen wollen, das ihnen vorschwebt wie:<br />
      <ul>
        <li>Skripte</li>
        <li>Webseiten</li>
        <li>Data Science + Maschinelles Lernen (wirklich beliebter Verwendungsfall für Python)</li>
        <li>schnelles Prototypisieren</li>
        <li>Automation</li>
      </ul>
  </details>
  <details>
    <summary>
    2. Java
    </summary>
      Obwohl es nicht den einfachsten Weg bereithält, gibt diese Sprache dir ein solides Verständnis von OOP (Objekt Orientierte Programmierung)<br />
      Dies ist keine offensichtliche Wahl, da die Sprache von Zeit zu Zeit komplex werden kann. Sie ist gut für Leute, die eine Herausforderung suchen, aber auch für Leute die Java für die Umsetzung von Projekten wie den folgenden verwenden möchten:<br />
      <ul>
        <li>Android Apps</li>
        <li>Backend Entwicklung</li>
        <li>OOP Vertiefung</li>
      </ul>
  </details>
  <details>
    <summary>
    3. Kotlin
    </summary>
      Die Sprache ist nicht so einfach wie Python, aber einfacher als Java.<br />
      Es ist sozusagen die moderne Reinterpretation von Java. Weniger Boilerplate-Code, einfacher zu erlernen, modernere (und andere) Funktionen.
      <br />
      Kotlin ist gut für Leute, die mit einer mittleren Schwierigkeit starten wollen und ist es exzellent für Projekte wie diese:<br />
      <ul>
        <li>Android Apps(<a href="https://developer.android.com/kotlin/first?hl=de">seit 7. Mai, 2019 ist Kotlin die offizielle Sprache für die Android Entwicklung</a>)</li>
        <li>Backend Entwicklung (moderner, oft als Migration von Java benutzt, da diese Interoperabilität haben (<a href="https://developer.android.com/kotlin/interop?hl=de">Java kann Kotlin Code aufrufen und umgekehrt</a></li>
        <li>OOP Vertiefungen(noch interessanter wie Kotlin dies handhabt)</li>
      </ul>
  </details>
  <details>
    <summary>
    4. JavaScript
    </summary>
      Diese Sprache kein ein bisschen komisch sein, vor allem die Logik kann einen etwas verwirren. Beispiel: <a href="https://www.youtube.com/watch?v=p-cEEnODLI8">JavaScript ist so logisch! - Herr Programmierer</a><br />
      Es ist eine sehr fokussierte Sprache (was damit zusammenhängen könnte, dass JavaScript oft als Websprache auftritt). <br />
      Es bietet trotzdem noch einfach Syntax und wertvolle Lernerfahrungen.Projekte, die du mit dieser Sprache umsetzen kannst:<br />
      <ul>
        <li>Webseiten</li>
        <li>Cross Platform Apps (über Frameworks)</li>
        <li>Skripte</li>
      </ul>
  </details>
</details>

_Das wäre das Ende der Anfangssprachen Sektion. Ich hoffe du hast ein Verständnis dafür gewonnen, was diese Sprachen dir ermöglichen und wurdest dabei unterstützt die deinen Bedürfnissen/Interessen entsprechende Sprache zu wählen._
