# Java course

## What is Java Space Tutorial Course?

The Java space tutorial course guides you step-by-step through the process of learning the programming language Java. It was made by the same person that wrote this curriculum.

Link will be coming soon.

## Why should you take the course?

Java is one of the most used enterprise languages out there. It's also listed as one of the starter languages in this curriculum. Due to its more complex nature it can be hard to find project-based tutorials for this languages like the "The Odin Project" for web related programming (HTML, CSS, JS/Ruby). That's why I've decided to move forward with creating a course around the programming language Java. One I currently get taught in school and will produce a course for in order to use the Feynman technique in a creative way which I think will also benefit others.

## Is Java worth the effort?

That depends on who you ask. Java isn't worth the effort for every use case but it has its strengths that make it really useful for certain use cases (Android apps, enterprise grade apps, backend migrations). As with every thing you learn, you should evaluate why you want to learn it and how you are going to use it (if your reason isn't just doing it for fun).

## Why shouldn't I choose another language?

I'm not going to convince you to learn a certain language. It depends entirely on your situation if this language is the right for you. If it isn't, just choose another.
