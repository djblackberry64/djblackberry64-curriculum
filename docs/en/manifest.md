# Reasoning, project state and future plans

## Reasoning

This project started as a simple tutorial project by me where I wanted to teach programming

_something which I saw was often saturated with text heavy content_

through self-shot photography and a narrative structure.

To remove as much friction as possible while still keeping customization I chose MkDocs.

## Project State

This project currently uses a static site generator called MkDocs with the Material for MkDocs theme. Both of these are commonly used on documentation sites.

This project is currently available in English (most recent) and German (a little behind).

## Future plans

This was an awesome start for my idea to teach programming through self-shot photography and a narrative.

In the future, I hope to utilize my web development capabilities to built a site/use a more customizable engine which allows me to optimize my workflow and have even more development options without restrictions.

I also plan on incorporating the user feedback I already received from various sources including but not limited to YSWS events.

I hope y'all could already get value out of this curriculum and are excited for more content as well as new possibilities to come!
